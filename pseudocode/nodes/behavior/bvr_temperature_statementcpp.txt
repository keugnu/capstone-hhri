Node:	temperature_statement (Speech response to temperature state)
Input:
	Textual/spoken statement asking for temperature.
	Float from temperature sensor.
Output:
	Audio stating temperature value as degrees Fahrenheit.
Steps:
	Initialization: begin subscribing to tpc_temperature (5Hz)
	1. Recognize input statement asking for temperature
	2. Convert temperature float value to string.
	3. Append "degrees Fahrenheit" to temperature string.
	4. Parse the temperature text statement to audio
	5. Return temperature audio to speaker.
