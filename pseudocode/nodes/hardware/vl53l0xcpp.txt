Node:	vl53l0x_pub (TOF Proximity Sensor)
Input:
	None
Config:
	int16 address
Output:
	float32 range //cm
Steps:
	1. Initialization: Send read request to I2C bus manager node (10Hz)
	2. Publish range to tpc_tracker
	3. if an asynchronous read request made then
		(a) send request to I2C bus manager
		(b) Return value from request to srv_proximity
